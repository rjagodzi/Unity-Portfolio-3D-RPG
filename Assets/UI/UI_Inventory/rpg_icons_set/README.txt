Set of 245 icons that I made for my game.

They are all 64x64 pixels and packed on 1024x1024 atlas.

For some icons I used images from Pixabay and Unsplash as base, and also some CC0 assets from OGA.

Not all icons are great, but I hope you find at least some of them useful :)
